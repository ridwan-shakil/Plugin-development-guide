Create draggable admin notes with checklists inside your WordPress dashboard. Easily organize important tasks or reminders.

== Description ==

Plugmint Draggable Notes For Admin lets you create and organize custom notes directly inside the WordPress admin dashboard.  
Each note is fully draggable and can include simple checklists for task management. Note items can be draggable from one note to another note.

Perfect for site managers, content editors, and teams who want a simple way to keep reminders and to-dos inside the admin panel.

**Key features:**

* Create unlimited notes
* Drag and drop notes anywhere on the dashboard
* Add checklist items inside notes
* Checklist items are also draggable and can be moved from one note to another note
* Choose note colors for better organization
* Show/hide notes by user (optional)
* Lightweight — no external libraries except jQuery UI
* Fully responsive and compatible with all admin themes

This plugin does not store data outside your website and does not communicate with external servers.

== Installation ==

1. Download the plugin ZIP file.
2. Upload it to **Plugins → Add New → Upload Plugin**.
3. Activate the plugin.
4. Go to **Dashboard → Admin Notes** to start adding notes.
5. Drag and organize notes directly on the dashboard.

== Frequently Asked Questions ==

= Where are the notes stored? =
All notes are stored inside the WordPress database under a custom post type. No external storage is used.

= Will the notes be visible to all admins? =
Only the person who created the note can see it by default. However, visibility can be set to Onlyme, all administrators, editors & above.

= Can I export or import notes? =
Currently no, but this feature may be added in a future update.

= Do the notes affect site performance? =
No. Notes are lightweight and only load in the admin area.
